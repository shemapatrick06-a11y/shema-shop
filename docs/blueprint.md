# **App Name**: TrendyThreads

## Core Features:

- Product Browsing: Allow users to browse and filter products with a responsive masonry grid layout.
- Product Details: Display detailed product information, zoomable images, size selection, and customer reviews.
- Shopping Cart: Enable users to add items to a shopping cart, adjust quantities, and apply promo codes.
- Checkout: Guide users through a streamlined checkout process with shipping, payment (including Google Pay), and order confirmation steps.
- Admin Dashboard: Provide an intuitive admin panel for managing products, orders, users, and analytics.
- Recommendation Engine: AI-powered tool to suggest related products based on viewing history and purchase behavior.
- Firebase Integration: Integrate with Firebase services including Authentication, Firestore database to store products, orders, users and use Firebase Hosting

## Style Guidelines:

- Primary color: Deep blue (#2c3e50) to evoke trust and style.
- Background color: Soft white for optimal readability.
- Accent color: Warm orange (#ff8c00) for buttons and interactive elements.
- Headline font: 'Playfair', a modern serif, for titles and headings.
- Body font: 'PT Sans', a humanist sans-serif, for body text and descriptions.
- Use clean, minimalist icons for navigation and user interactions.
- Implement subtle animations for smooth transitions and user feedback, like a confetti burst on order confirmation.
